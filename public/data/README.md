# Data sources & licenses

The snapshot files in this directory come from several sources, each
with its own license. Code in this repository is MIT; data is NOT —
data inherits its source's license regardless.

## OpenStreetMap-derived (ODbL-1.0)

© OpenStreetMap contributors, licensed under the
[Open Database License (ODbL) v1.0](https://opendatacommons.org/licenses/odbl/1-0/).
ODbL is share-alike: any publicly distributed derivative database must
also be ODbL.

- `addr_index.json` — addresses + POIs from Overpass
- `routing_graph.json` — OSM way topology, augmented with SDOT attribute joins
- `seattle_polygon.geojson` — Seattle boundary from Nominatim

## USGS 3D Elevation Program (public domain)

US Government work, public domain under 17 U.S.C. §105.

- `contours.geojson` — 25-ft contours extracted from USGS 3DEP DTM

## Seattle SDOT (open municipal data)

Published by the Seattle Department of Transportation via
[Seattle GeoData](https://data-seattlecitygis.opendata.arcgis.com/).

- `beacons.geojson`, `bicycle_racks.geojson`, `bike_facilities.geojson`,
  `bike_plus_network.geojson`, `bike_signs.geojson`,
  `community_centers.geojson`, `crosswalks.geojson`, `libraries.geojson`,
  `light_rail_stations.geojson` (via SDOT, originally Sound Transit),
  `multi_use_trails.geojson`, `parks_restrooms.geojson`,
  `seattle_streets.geojson`, `signals.geojson`, `stop_signs.geojson`,
  `traffic_circles.geojson`

## King County (open municipal data)

Published by [King County GIS](https://gisdata.kingcounty.gov/).

- `kc_regional_trails.geojson`

## Build artifact

- `version.json` — PWA precache manifest generated by
  `scripts/build_data_manifest.py`; covered by the repo's MIT license.
